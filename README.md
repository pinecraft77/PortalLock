# PortalLock
![PortalLock](PortalLock.png)
![Minecraft](https://img.shields.io/badge/Minecraft-Fabric-brightgreen)
![Version](https://img.shields.io/badge/Version-v1.2.0-blue)
![License](https://img.shields.io/badge/License-All%20Rights%20Reserved-red)

A Fabric mod that locks Nether and End portals using configurable key items.

Players must possess (and optionally consume) the required key item before entering protected portals.

---

# ✨ Features

- 🔑 Lock Nether Portals
- 🌌 Lock End Portals
- ⚙️ Configurable key items
- 📦 Configurable required item count
- 🎵 Custom sounds
- 💬 Custom messages
- 🌍 Multi-language support
- 🔄 `/pl reload` command
- 🚤 Boat support
- ⛏️ Minecart support
- 🐾 Animal mob riding support (v1.2.0+)
- 🎨 Color code support
- 🛡️ Creative mode support

# ⚙️ Configuration

PortalLock can be fully customized.

- Required key item
- Required item count
- Consume item on use
- Sounds
- Messages
- Overlay messages
- Chat messages
- Language
- Color formatting
